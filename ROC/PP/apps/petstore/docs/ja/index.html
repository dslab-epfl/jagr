Java Pet Store Demo 1.3.1_02

Java Pet Store Demo は、Sun Microsystems 社の Java Softwareの Web サイトの Java BluePrints プログラムから提供されているサンプルアプリケーションです。

このサンプルアプリケーションは、J2EE プラットフォームの機能を利用して、堅牢で、スケーラブルで、移植性があり、かつ保守の容易な e-Business アプリケーションを開発する方法を説明するためのものです。完全なソースコードとドキュメントが同梱されているので、J2EE テクノロジを試してみることができ、それらのテクノロジを利用して独自のエンタープライズソリューションを効率的に開発する方法を学ぶことができます。このデモのバージョン 1.3.1_02 では、J2EE プラットフォームで Web サービステクノロジを利用する方法も紹介しています。

Java BluePrints for the Enterprise プログラムについて

Java BluePrints for the Enterprise プログラムでは、J2EE プラットフォームでのアプリケーションプログラミングモデルを定義します。実社会のアプリケーションシナリオのためのベストプラクティスのガイドラインや構造上の推奨事項を提供することにより、開発者が J2EE テクノロジを利用して、移植性があり、スケーラブルで、かつ堅牢なアプリケーションを作成できるようにします。

Java BluePrints for the Enterprise プログラムと Java Pet Store Demo については、『Java2 Platform, Enterprise Edition アプリケーション設計ガイド 第 2 版』で紹介されています。このベストセラー書は、お近くの書店や、DigitalGuru および Amazon.com などの Web サイトで購入できます。

最新の内容については、http://java.sun.com/blueprints/ の Java BluePrints の Web サイトを参照してください。
