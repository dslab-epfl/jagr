リリースノート

Java Pet Store Demo 1.3.1_02

このリリースノートでは、Java Pet Store Demo 1.3.1_02 に関する既知の問題について説明します。

1. 管理クライアントの問題

環境 : Java Pet Store Demo 1.3.1_02、Java 2 SDK, Enterprise Edition v1.4 Beta、Java Web Start v1.2 または Java Web Start v1.0.1、および Internet Explorer

問題 : Java Pet Store Demo 1.3.1_02 が J2EE リファレンス実装 1.4 Beta 上に配備されているとき、Java Web Start で管理クライアントを起動できません。現在ダウンロードできる Java Web Start (Java Web Start v1.2 または Java Web Start v1.0.1_02) のどちらのバージョンでも、Windows 上の Internet Explorer で管理クライアントを起動できません。

2. RI1.4 Beta での国際化

環境 : Java Pet Store Demo 1.3.1_02 および Java 2 SDK, Enterprise Edition v1.4 Beta

問題 : RI Beta バージョン 1.4 には国際化に関する既知の問題があります。このため、Pet Store の Storefront の日本語版と中国語版は、1.4 Beta RI では正常に機能しない場合があります。
