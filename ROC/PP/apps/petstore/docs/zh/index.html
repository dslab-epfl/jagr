Java Pet Store Demo 1.3.1_02

Java Pet Store Demo 是按照 Sun Microsystems 的 Java Software中的Java BluePrints计划为您提供的一个示范应用程序。

这个示范应用程序演示了如何利用 J2EE 平台开发强有力的、可升级的、可移植的、并易于维护的电子商务应用程序。该应用程序随附全部源代码和文档，您可以尝试应用 J2EE 技术并学习如何有效地使用这些技术构成自己的企业解决方案。1.3.1_02 演示版还展示了如何在 J2EE 平台上启用 Web 服务技术。

有关 Java BluePrints for the Enterprise 计划

Java BluePrints for the Enterprise（企业 Java 蓝图）计划定义了 J2EE 平台的应用程序编程模式。针对现实应用情形，它提供了最好的实际操作指导和架构建议，使得开发者能够应用 JZEE 技术生成可移植的、可升级的、强有力的应用程序。

在 Designing Enterprise Applications with the J2EE Platform一书的第二版中详细介绍了有关 Java BluePrints for the Enterprise 计划和 Java Pet Store Demo 的知识。这本畅销书可在任何地方的书店买到，包括通过网站 DigitalGuru 和 Amazon.com。
