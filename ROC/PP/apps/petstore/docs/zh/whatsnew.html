Java Pet Store Demo 1.3.1_02 中的新增功能

下面是自 1.3.1 版发布以来对 Java Pet Store Demo 应用程序所做的改动：
  • 并入了 BluePrints 开发团队提出的Naming Conventions for Enterprise Applications（企业应用程序的命名规范）中的一些命名规范
  • 增加了简体中文本地化版本的 PetStore 前端
  • 对前一版本的一些程序错误进行了修正和清除
