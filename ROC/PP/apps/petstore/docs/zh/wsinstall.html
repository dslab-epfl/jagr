在演示版中启用 Web 服务

这部分内容介绍如何在 Java Pet Store Demo 中启用 Web 服务功能。要在演示版中启用 Web 服务功能，需具备 Java 2 SDK 企业版 (J2EETM SDK)，v1.3.1。如果想在 Java 2 SDK 企业版 (J2EETM SDK) v1.4 上部署 Web 服务，请参考 BluePrints 网站。必须首先安装此处说明的演示版的基本版本。

系统要求：

需要下载和安装如下的其它软件：

    1. Java Web Services Developer Pack，v1.0_01：单击此处下载

必需的环境变量：

需要设置一个额外的环境变量：

    JWSDP_HOME 必须指向 Java Web Services Developer Pack (JWSDP) 安装位置的根目录。例如，如果将 JWSDP 安装在/usr/local/java/jwsdp-1_0_01 路径下面，那么这个路径就是JWSDP_HOME 的值。在文档的其余部分我们将用 <jwsdp.home> 指代这个路径。

安装演示版

要安装具有 Web 服务功能的演示版，按如下步骤操作：

运行脚本将 Web 服务技术集成到 J2EE 平台

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令：

cd <jwsdp.home>/bin
sh jwsdponj2ee.sh $J2EE_HOME
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令：
cd <jwsdp.home>\bin
jwsdponj2ee.bat %J2EE_HOME%

启动 Cloudscape 数据库。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

cd $J2EE_HOME/bin
cloudscape -start
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
cd %J2EE_HOME%\bin
cloudscape.bat -start
启动 J2EE 服务器。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

cd $J2EE_HOME/bin
j2ee -verbose
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
cd %J2EE_HOME%\bin
j2ee.bat -verbose
    转到启用 Web 服务的 petstore 目录下。

    如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

    cd <petstore.home>/webservices
    如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
    cd <petstore.home>\webservices
    运行安装脚本来部署应用程序。

    如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

    sh setup.sh deploy
    如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
    setup.bat deploy
安装演示版后，就可以开始使用了。
