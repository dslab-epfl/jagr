安装演示版

这部分内容介绍如何安装 Java Pet Store Demo

系统要求：

在安装演示版前，需要安装下列软件：
    1. Java 2 SDK 标准版 (J2SETM SDK)，v1.4.1：单击此处下载

    2. Java 2 SDK 企业版 (J2EETM SDK)，v1.3.1：单击此处下载

    或

    Java 2 SDK，企业版 (J2EE TM SDK)，v1.4 Beta 版：单击此处下载

必需的环境变量：

必须按如下方式设置环境变量 JAVA_HOME 和 J2EE_HOME ：
    JAVA_HOME 必须指向 J2SE SDK 安装位置的根目录。例如，如果将 J2SE SDK 安装在 c:\j2sdk1.4.1 下面，那么这个路径就是JAVA_HOME的值。

    J2EE_HOME 必须指向 J2SE SDK 安装位置的根目录。例如，如果将 J2EE SDK 安装在/usr/local/java/j2sdkee1.3.1 下面，那么这个路径就是 J2EE_HOME 的值。

同样，您的路径必须包括目录 %JAVA_HOME%\bin（Window 系统）或目录 $JAVA_HOME/bin（Unix 系统）。否则，一些程序编译任务将无法工作。

安装演示版

在 Java 2 SDK 企业版 (J2EE TM SDK)，v1.4 Beta 版下安装演示版本

    将发行包（petstore-1_3_1_02.zip）解压到任何目录下面。

    将创建一个名为 petstore1.3.1_02 的目录。本文档的其余部分将用 <petstore.home> 指代这个目录。

编辑 <j2ee.home>/lib/security/server.policy 文件
在文件末尾处插入下列语句：

grant {
permission java.io.FilePermission "${com.sun.enterprise.home}${/}lib${/}endorsed${/}xalan.jar", "read";
};

启动 Cloudscape。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

cd $J2EE_HOME/bin
cloudscape -start
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
cd %J2EE_HOME%\bin
cloudscape.bat -start
启动 J2EE 服务器。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

cd $J2EE_HOME/bin
j2ee -verbose
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
cd %J2EE_HOME%\bin
j2ee.bat -verbose

    运行安装脚本以便安装和配置消息服务和数据库资源。

    如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

    cd <petstore.home>
    sh setup.sh
    如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
    cd <petstore.home>
    setup.bat

终止 Cloudscape。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

cd $J2EE_HOME/bin
cloudscape -stop
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
cd %J2EE_HOME%\bin
cloudscape.bat -stop
终止 J2EE 服务器。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令：

如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令：

启动 Cloudscape。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

cd $J2EE_HOME/bin
cloudscape -start
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令：

启动 J2EE 服务器。

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

cd $J2EE_HOME/bin
j2ee -verbose
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
cd %J2EE_HOME%\bin
j2ee.bat -verbose
    运行安装脚本来部署应用程序。

    如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令： 

    cd <petstore.home>
    sh setup.sh deploy
    如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令： 
    cd <petstore.home>
    setup.bat deploy
