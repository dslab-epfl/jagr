发行说明

Java Pet Store Demo 1.3.1_02

本发行说明描述了 Java Pet Store Demo 1.3.1_02、Java 2 SDK 企业版、1.4 Beta 版本在 Windows 系统上存在的问题。 . 仅在 Internet Explorer 上存在问题。

1. Admin 客户端问题

环境：Java Pet Store Demo 1.3.1_02、Java 2 SDK 企业版 1.4 Beta版本、Java Web Start v1.2 或 Java Web Start v1.0.1 和 Internet Explorer

问题：当 Java Pet Store Demo 1.3.1_02 部署在 J2EE 参考实现 1.4 Beta 版上时，Web Start 不能启动管理客户端程序。

现在供下载的 Java Web Start 的两个版本（Java Web Start v1.2 或 Java Web Start v1.0.1_02）在 Windows 的 Internet Explorer 下都不能启动管理客户端程序。

2. 参考实现1.4 Beta 中的国际化

环境：Java Pet Store Demo 1.3.1_02, Java 2 SDK, 企业版 v1.4 Beta

问题：在 1.4 参考实现 Beta 版本中存在已知的国际化问题。所以 PetStore 商店的日文和中文前端也许无法在 1.4 Beta 参考实现上正常运行
