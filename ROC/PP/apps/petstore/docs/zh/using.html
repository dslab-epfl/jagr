演示版概览

Java Pet Store Demo 包含下列高级组件：

  • Storefront 展现了 Web 前端的主用户界面，客户使用 Storefront 提交宠物订单。

  • 订购处理中心 (OPC) 接收来自 Storefront 的订单。

  • “供应商”从库存完成来自 OPC 的订单并向 OPC 开具发票。

  • Admin 在 JFC/Swing 前端提供管理员界面。 管理员使用 Admin 来检查“未决”订单并批准或拒绝。

使用 Storefront

访问 Storefront

使用浏览器访问下列 URL：

http://localhost:8000/petstore

如果安装了 Java Web 服务开发者包 (Java Web Services Developer Pack)，则应访问如下 URL：

http://localhost:8080/petstore

创建新用户

如果这是您第一次访问这个商店，在购买前需创建一个新用户。

  1. 单击主页上的“登录进入”。

  2. 填写标题为“我愿意注册一个帐户”的表单，然后单击“创建新帐户”。

  3. 输入帐户信息，单击“提交”。

提交订单

例如，要订购一只牛头犬，请按如下步骤进行：

  1. 在主页上，选择“狗”。

  2. 单击“牛头狗”。

  3. 单击“成年雄性牛头犬”行中的“添加到购物车”。

  4. 单击“结账”。

    会出现一个包含帐单信息和运送信息的表单。为了方便起见，已填好了这个表单 。

  5. 单击“提交”。

完成订购后，Storefront 以异步方式将订单信息送到 OPC。因为订单数额不超过 500 美元，OPC 会自动批准这个订单。

使用管理员客户端程序

打开管理员客户端程序

使用 Java 管理员客户端对订单进行审查和修改。

使用浏览器访问下列 URL：

http://localhost:8000/admin/

如果安装了 Java Web 服务开发者包，则应访问如下 URL：

http://localhost:8080/admin

要启动“管理员胖客户端”界面，单击主页上的“启动胖客户端”。通过使用 Java Web Start 启动客户端程序。

客户端界面出现时，会显示两个窗口，分别显示“未决”和“已决”订单。前面部分提交的牛头狗订单将显示在“已决”订单中，因为这个订单的数额未超过 500 美元，所有不超过 500 美元的订单都被自动批准。

如果已经在演示版中启用 Web 服务，也可以利用该 Java 客户端程序的独立版与使用 Web 服务的服务器通信。要运行这个客户端程序，请按如下步骤进行：

如果使用的是 Unix 操作系统，打开 shell 命令提示符并输入下述命令：

cd <petstore.home>/webservices
sh adminwsclient.sh localhost 8080
如果使用的是 Windows 操作系统，打开命令行提示符并输入下述命令：
cd <petstore.home>\webservices
adminwsclient.bat localhost 8080

查看“已决”订单

“已决”订单视图面板显示“已决”订单。所有的这些订单都已被批准、拒绝或已执行（完成）。要加载菜单，单击“刷新”按钮。

要为“供应商”创建或编辑库存目录，请参阅管理供应商程序。

批准/拒绝订单

利用“处理‘未决’订单”面板来批准或拒绝“未决”订单。如果一个订单的总金额超过 500 美元，它就显示为“未决”状态。

批准或拒绝一个订单：

  1. 单击“处理‘未决’订单”标签。

  2. 单击该订单行。

  3. 单击其“状态”单元。

  4. 选择一个新状态。

您可批准或拒绝不限数目的订单。要保存所做的改动，单击“执行”。要查看所做的改动，单击“刷新”按钮。

管理供应商程序

打开供应商客户端

利用一个简单的 Web 界面来管理供应商及其库存。在使用“供应商”界面之前，首先要确保退出“管理员客户端程序”。

使用浏览器访问下列 URL：

http://localhost:8000/supplier/

如果安装了 Java Web 服务开发者包，则应访问如下 URL：

http://localhost:8080/supplier/

查看并编辑库存

要查看和编辑库存，单击主页屏幕上的“显示库存”。

一个表格显示每项库存的数量。要编辑货项的数量，应编辑此货项条目中的“新数量”单元，再单击“更新”框。做完所有改动后，单击“提交”将编辑结果送到数据库中。

只要更新数据库，“供应商”程序都会通知 OPC 有关库存的变动。OPC 检查这些变动以确定它现在是否能完成以前不能完成的订单。如果它可以完成这个订单，就会在订单上标记为已完成。
