デモのインストール

ここでは、Java Pet Store Demo をインストールする方法について説明します。

システム要件

デモをインストールする前に、次のソフトウェアをインストールしておく必要があります。
    1. Java 2 SDK, Standard Edition (J2SETM SDK), v1.4.1: ダウンロード

    2. Java 2 SDK, Enterprise Edition (J2EETM SDK), v1.3.1: ダウンロード


必要な環境変数

環境変数 JAVA_HOME および J2EE_HOME を次のように設定する必要があります。
    JAVA_HOME には、J2SE SDK のインストール先のルートディレクトリを指定する必要があります。たとえば、J2SE SDK を c:\j2sdk1.4.1 にインストールした場合は、そのパスを JAVA_HOME の値として設定する必要があります。

    J2EE_HOME には、J2EE SDK のインストール先のルートディレクトリを指定する必要があります。たとえば、J2EE SDK を /usr/local/java/j2sdkee1.3.1 にインストールした場合は、そのパスを J2EE_HOME の値として設定する必要があります。

また、PATH 変数には、ディレクトリ %JAVA_HOME%\bin (Windows の場合) または $JAVA_HOME/bin (UNIX の場合) を含める必要があります。変数を適切に指定しないと、ビルド作業に失敗することがあります。

デモのインストール

    配布バンドル (petstore-1_3_2.zip) を任意のディレクトリに解凍します。               

    petstore1.3.2 というディレクトリが作成されます。このドキュメントの以降の部分では、このディレクトリを <petstore.home> と表記します。

    設定スクリプトを実行して、メッセージサービスとデータベースリソースをインストールして構成します。

    UNIX を使用している場合は、シェルプロンプトを開き、次のコマンドを入力します。 

    cd <petstore.home>
    sh setup.sh
    Windows を使用している場合は、コマンドプロンプトを開き、次のコマンドを入力します。 
    cd <petstore.home>
    setup.bat

Cloudscape を起動します。

UNIX を使用している場合は、シェルプロンプトを開き、次のコマンドを入力します。 

cd $J2EE_HOME/bin
cloudscape -start
Windows を使用している場合は、コマンドプロンプトを開き、次のコマンドを入力します。 
cd %J2EE_HOME%\bin
cloudscape.bat -start
J2EE サーバを起動します。

UNIX を使用している場合は、シェルプロンプトを開き、次のコマンドを入力します。 

cd $J2EE_HOME/bin
j2ee -verbose
Windows を使用している場合は、コマンドプロンプトを開き、次のコマンドを入力します。 
cd %J2EE_HOME%\bin
j2ee.bat -verbose
    設定スクリプトを実行して、アプリケーションを配備します。

    UNIX を使用している場合は、シェルプロンプトを開き、次のコマンドを入力します。 

    cd <petstore.home>
    sh setup.sh deploy
    Windows を使用している場合は、コマンドプロンプトを開き、次のコマンドを入力します。 
    cd <petstore.home>
    setup.bat deploy


デモをインストールした後は、Pet Store の使用を開始できます。
