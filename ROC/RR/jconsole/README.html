A JMX Console for JBoss

This console, unlike the one that is provided as part of JBoss, provides a good summary of JBoss characteristics with much less clutter.  It also gives you the ability to do pretty much everything the JBoss JMX console can do.

To deploy:
  1. Copy the jconsole.war file into $JBOSS_HOME/server/default/deploy
    (you should see JBoss automatically deploy it).
  2. Point your browser to http://localhost:8080/jconsole/
    (it will take a while before you get the first page, but from there on it's pretty quick).

Downloaded from Coldbeans.
