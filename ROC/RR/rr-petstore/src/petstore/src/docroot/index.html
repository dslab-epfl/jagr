JavaTM Pet Store Demo

This application is a fictitious e-commerce site known as the Java Pet Store Demo. Although the application may look similar to many other e-commerce sites available today on the Web, the underlying technology is the J2EETM platform making it scalable, secure, and platform independent.

This specific e-commerce site specializes in selling pets. The application was designed in a general way so that the catalog of products can be customized for other types of products with little or no change needed in the application code.

Please refer to the JavaTM 2 Platform, Enterprise Edition Blueprints for more information about this application. You can access them at http://java.sun.com/j2ee/blueprints/.
